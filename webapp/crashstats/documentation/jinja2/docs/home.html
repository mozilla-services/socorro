Need help?

If you found a bug, please file a bug report.

If you want to add support for a new product or remove support for an existing product, see our Confluence page.

If you want to change signature generation, see Socorro signature generation documentation for documentation on how Socorro's signature generation works and how to request and make changes to it.

If you want to add support for new crash annotations, see Socorro crash annotations documentation for documentation on how to do that and how to request support for those annotations in Crash Stats and Telemetry.

For any other needs, see our Confluence page.

Accounts on Crash Stats

Sign up to create an account if you require one of the following:

  • Protected data access. See protected data access documentation for details.
  • API tokens.

If you don't need one of those things, then signing up for an account doesn't help you.

Crash Stats accounts are created the first time you log in. You can log in using the Log in link in the upper right hand corner.

If you are a Mozilla employee, please log in using your LDAP account.

Socorro developer documentation

See Socorro code/developer documentation for documentation on the code that runs Socorro including how to make changes, submit pull requests, and setting up a local development environment.
