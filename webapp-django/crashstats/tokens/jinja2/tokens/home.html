How To Use These

When using the API you must supply these tokens as a header called Auth-Token.

Here's an example:

curl -H "Auth-Token: 58af2acef8a74dbca9580e2bb8ba9c9a" {{ absolute_base_url }}{{ url('api:model_wrapper', 'RawCrash') }}?uuid=fb27d0f3-db8e-4835-9311-288bb0170829

Or, if you prefer Python:

import requests

headers = {'Auth-Token': '58af2acef8a74dbca9580e2bb8ba9c9x'}
url = '{{ absolute_base_url }}{{ url('api:model_wrapper', 'RawCrash') }}?uuid=fb27d0f3-db8e-4835-9311-288bb0170829'
response = requests.get(url, headers=headers)
print response.json()
